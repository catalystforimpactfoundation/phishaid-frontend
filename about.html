About PhishAID

PhishAID is an academic cybersecurity project focused on detecting phishing websites using a transparent, rule-based detection engine.

The project is developed as part of an academic curriculum and is aligned with national-level AI challenge problem statements related to phishing detection.

Unlike black-box machine learning systems, PhishAID emphasizes explainability, interpretability, and early-stage risk detection, making it suitable for academic evaluation and security awareness.
