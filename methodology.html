Detection Methodology

PhishAID evaluates URLs using a predefined set of security rules derived from known phishing characteristics.

Rule Categories

  • URL lexical analysis
  • IP-based URLs
  • Suspicious keywords
  • HTTPS and certificate checks
  • URL length and entropy

Each rule contributes to a cumulative score which determines the final verdict.
