Detection Methodology

Overview

PhishAID evaluates URLs using a set of predefined security rules. Each rule checks for known phishing indicators.

Rule Categories

  • URL structure analysis
  • IP-based URLs
  • Suspicious keywords
  • HTTPS & certificate checks
  • Length and entropy analysis

Scoring Mechanism

Each triggered rule contributes to a cumulative risk score. The final verdict is derived from the total score.

Explainability

Every verdict is accompanied by human-readable warnings, making the system suitable for academic interpretation.
