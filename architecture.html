System Architecture

High-Level Design

PhishAID follows a serverless, edge-hosted architecture.

  • User Browser
  • Cloudflare CDN & SSL
  • Cloudflare Pages (Frontend)
  • Rule-based Detection Engine

Request Flow

  1. User submits a URL
  2. Frontend sends request to detection logic
  3. Rules are evaluated
  4. Verdict and warnings are displayed
