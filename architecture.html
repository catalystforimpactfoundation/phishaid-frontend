System Architecture

PhishAID uses a lightweight, serverless architecture designed for scalability and ease of deployment.

  • User Interface (HTML/CSS)
  • Rule Engine (JavaScript)
  • Static Hosting (Cloudflare Pages)
  • CDN & Security (Cloudflare)

This architecture minimizes maintenance while ensuring high availability.

PhishAID follows a serverless, edge-hosted architecture using Cloudflare services.

  1. User submits URL via browser
  2. Request passes through Cloudflare CDN
  3. Rule engine evaluates URL
  4. Verdict and warnings returned
