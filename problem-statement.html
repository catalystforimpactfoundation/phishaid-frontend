Project Origin & Problem Statement

Source of the Idea

PhishAID is inspired by Problem Statement PS-02 from a national AI challenge focused on detecting phishing domains and URLs targeting critical sector entities.

Problem Definition

Phishing attacks exploit user trust by creating deceptive URLs, domains, and web interfaces that impersonate legitimate organizations.

Academic Motivation

  • Early detection of suspicious URLs
  • Explainable decision-making
  • Independence from proprietary threat feeds

Outcome

PhishAID translates the problem statement into a lightweight, rule-based web application suitable for academic evaluation.
