Problem Statement

Phishing attacks continue to be one of the most prevalent cybersecurity threats, causing financial loss, identity theft, and data breaches.

This project is inspired by AI Challenge Problem Statement (PS-2), which focuses on detecting phishing websites using intelligent techniques.

The objective is to design a system that detects phishing attempts while maintaining transparency and explainability.
