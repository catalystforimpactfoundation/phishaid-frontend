Project Origin & Problem Statement

PhishAID is inspired by Problem Statement PS-02 from a national-level AI Challenge addressing phishing detection targeting critical sector entities.

The challenge highlights the need for early detection of malicious domains, look-alike URLs, and deceptive web links before large-scale exploitation.

This project translates the challenge requirements into a practical, rule-based web application suitable for academic evaluation.
