Phishing Detection Rules

  • URL contains IP address
  • Excessive URL length
  • Suspicious keywords
  • Missing HTTPS
  • Multiple subdomains
  • Encoded characters
  • High entropy strings
  • Shortened URLs
  • Unusual port numbers
  • Look-alike domain patterns

These rules are heuristic-based and designed for explainable phishing detection.
